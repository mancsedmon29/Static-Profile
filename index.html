Capstone 1: Static Profile Tutorial

Build your first HTML & CSS profile page in 4 simple steps

Live Preview
Tutorial Steps

Step 1: HTML Structure

Create your basic HTML file with header, sections for About, Hobbies, and Contact.

index.html 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Static Profile</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <header class="profile-header">
            <h1>Juan Dela Cruz</h1>
            <p class="tagline">Aspiring Web Developer</p>
        </header>
    </div>
</body>
</html>

Step 2: CSS Styling

Add styles to make your profile look clean. Use Poppins font and a gradient background.

style.css 
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Poppins', sans-serif;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    padding: 20px;
}

.container {
    max-width: 800px;
    margin: 0 auto;
    background: white;
    border-radius: 15px;
}

.profile-header {
    background: #2d3748;
    color: white;
    text-align: center;
    padding: 40px 20px;
}

Step 3: Hobbies Section

Create hobby cards using CSS Grid. Add hover effect for interactivity.

HTML & CSS 
<section class="hobbies">
    <h2>Hobbies & Interests</h2>
    <div class="hobby-list">
        <div class="hobby-card">
            <h3>🎮 Gaming</h3>
            <p>Valorant, Mobile Legends</p>
        </div>
    </div>
</section>

/* CSS */
.hobby-list {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 20px;
}

.hobby-card {
    background: #f7fafc;
    padding: 20px;
    border-radius: 10px;
    border-left: 4px solid #667eea;
}

.hobby-card:hover {
    transform: translateY(-5px);
    transition: 0.3s;
}

Step 4: Responsive Design

Make your site mobile-friendly using media queries. Stack cards on small screens.

style.css 
@media (max-width: 600px) {
    section {
        padding: 20px;
    }
    
    .hobby-list {
        grid-template-columns: 1fr;
    }
    
    .profile-pic {
        width: 120px;
        height: 120px;
    }
}
